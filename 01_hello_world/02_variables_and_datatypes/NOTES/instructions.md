2. operators in C
| Operator | Use            | Example |
| -------- | -------------- | ------- |
| `+`      | Addition       | `a + b` |
| `-`      | Subtraction    | `a - b` |
| `*`      | Multiplication | `a * b` |
| `/`      | Division       | `a / b` |
| `%`      | Modulus        | `a % b` |

RATIONAL OPERATORS:
used for comparisons (return true/flase -in C, 1 measn true and 0 means flase):
| Operator | Example  | Meaning          |
| -------- | -------- | ---------------- |
| `==`     | `a == b` | equal to         |
| `!=`     | `a != b` | not equal to     |
| `>`      | `a > b`  | greater than     |
| `<`      | `a < b`  | less than        |
| `>=`     | `a >= b` | greater or equal |
| `<=`     | `a <= b` | less or equal    |

LOGICAL OPERATORS :
used to combine conditions:
| Operator | Meaning | Example          |    |         |   |         |
| -------- | ------- | ---------------- | -- | ------- | - | ------- |
| `&&`     | AND     | `a > 0 && b > 0` |    |         |   |         |
| \`       |         | \`               | OR | \`a > 0 |   | b > 0\` |
| `!`      | NOT     | `!(a > 0)`       |    |         |   |         |
